phpmotors Database Leveling

What?

In week 03 you created the phpmotors database and imported a SQL file to create the inventory and carclassification tables. The clients table you created from scratch. The problem is that many of us are novices and may not have created the clients table completely or correctly.

This short activity is meant to fix that by making sure that all of our phpmotors tables are the same ("leveled") before we begin using them in earnest.

phpmotors SQL File

What I didn't tell you earlier is that the code to create the clients table is already in the downloadable SQL file that you used to create the other two tables, just commented out. We will uncomment the code and import the file again to make sure all of our phpmotors databases are the same.

Prepare the File

  1. Download the phpmotors SQL file again.
  2. Unzip the file.
  3. Store the phpmotors-db.sql file into your sql folder (created in week 03), for possible future use.
  4. Launch VS Code.
  5. Find and open the phpmotors-db.sql file.
  6. Scroll down to line 112 or so.
  7. Uncomment the following lines:
    • Lines 115 through 124
    • Lines 129 and 130
    • Lines 135 and 136

    To remove a comment, delete the -- on the left of the line or if using VS Code, highlight the lines and hold down Command + forward slash "/" on mac or Control + forward slash "/" on windows to undo a comment or add a comment. This key combination is a comment toggle.

  8. When done check that there are no errors (red icons or underlines) shown in the file!

Update the Database

  1. Start the Apache and MySQL servers.
  2. In a browser window, open the PHPMyAdmin tool.
  3. Click the phpmotors database (on the left or in the "Databases" tab).
  4. Click the "Import" tab.
  5. Click the "Choose File" button.
  6. Navigate to the phpmotors-db.sql file and click the "Open" button in the finder window.
  7. At the bottom of the Import window, click the "Go" button.
  8. If everything is successful you should see a green bar and on the left you should see the three tables listed beneath the phpmotors database.
  9. That's it! Everyone who does this will have the exact same tables, table structures and data.
  10. Be sure to shut down your servers when you are done working with them.
