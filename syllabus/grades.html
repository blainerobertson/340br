Class Grades

Objectives

Grading for this class is competency-based and depends on your demonstrated mastery of the course objectives (6 total) shown below:

You will demonstrate both the knowledge and skills to create dynamic web sites using:

  1. principles introduced in Web Frontend Development I - WDD 230;
  2. variables, arrays, functions, and control structures of the language.
  3. common design patterns while developing web applications.
  4. database(s) for storing data, while handling CRUD interactions and file storage.
  5. data validation (client-side and server-side) appropriate to the task;
  6. problem solving through independent study, working effectively with others, communicating clearly, fulfilling assignments and meeting deadlines.

Grade Scale

To pass an objective you must earn 80% of the points available for the objective. The grading scale (based on objectives passed) is shown below:
