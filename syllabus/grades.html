Class Grades

Expectations

All work is expected to be: 1) submitted on time, 2) complete, 3) fully-operational, and 4) follow best practice. As an upper-division course, it is expected that your work begins to resemble professional-level work. This requires you to stop thinking like a student, and start thinking like a professional.

Objectives

Grading for this class is competency-based and depends on your demonstrated mastery of the course objectives shown below. The grade book summarizes your preformance, by objective, at the very bottom.

  1. Enforce current standards of validity & best practice in the web frontend.
  2. Use variables, arrays, functions, and control structures on the server backend.
  3. Practice common design patterns while developing web applications.
  4. Implement databases during data / file storage and CRUD interactions.
  5. Validate data (client-side and server-side) appropriate to the task.
  6. Solve problems through independent study, working effectively with others, communicating clearly, fulfilling assignments and meeting deadlines.

Grade Scale

To pass an objective you must earn 80% of the points available for the objective. The grading scale (based on objectives passed) is shown below:
